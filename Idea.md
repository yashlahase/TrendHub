# TrendHub – Fashion E-commerce Website

## 📌 Project Overview
**TrendHub** is a modern fashion e-commerce website where users can explore trending clothing and accessories, view product details, add items to the cart, and place orders easily.  
It provides a smooth shopping experience with category-based browsing, search, and order tracking.

---

## ❓ Problem Statement
Many fashion shopping platforms are either too complex, slow, or not beginner-friendly.  
TrendHub aims to provide a **clean, fast, and user-friendly** shopping experience with essential e-commerce features.

---

## 🎯 Goals & Objectives
- Build a fully functional fashion e-commerce website with a professional UI.
- Provide easy navigation and smooth product discovery.
- Implement secure user authentication and order management.
- Create an admin panel to manage products and orders.

---

## ✨ Key Features

### 👤 User Features
- User Signup & Login (**JWT Authentication**)
- Browse products by category:
  - Men
  - Women
  - Kids
  - Accessories
- Search products by name
- Filter products by:
  - Category
  - Price range
- Product details page with:
  - Images
  - Price
  - Description
  - Size options (S/M/L/XL)
- Add to Cart / Remove from Cart
- Checkout & place order
- View order history

---

### 🛠️ Admin Features
- Add new products
- Update product details (price, stock, category, images)
- Delete products
- View and manage orders

---

## 👥 Target Users
- Students and young users looking for a simple fashion shopping platform
- Admin/sellers who want an easy system to manage products

---

## 🧰 Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **Authentication:** JWT + bcrypt
- **Styling:** Tailwind CSS / CSS

---

## ✅ MVP (Minimum Viable Product)
The first working version will include:
- Authentication (Register/Login)
- Product listing page
- Product details page
- Cart functionality
- Checkout page
- Order placement
- Order history page
- Basic admin product management
